# Quick TLDR
Do you want the ability to speak to the dead?
Look no further! Dead players can now speak though ouija boards.

# Features
Adds `/ouija <message>` to be used with a ouija board that can be found.

Shortcut messages included:
- y / yes
- n / no
- bye / goodbye
